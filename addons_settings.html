Ustawienia i Modyfikacje Rozszerzeń

UWAGA: Kruk Browser (oparty na Electronie) nie obsługuje natywnej instalacji plików *.crx. Rozszerzenia muszą być ładowane ręcznie jako rozpakowane foldery przez konsolę.

Jak ręcznie załadować rozszerzenie:

1. Pobierz rozszerzenie z Chrome Web Store i je rozpakuj.
2. Otwórz to okno za pomocą klawisza **F12** (lub Ctrl+Shift+I).
3. Użyj następującego kodu w konsoli, podając ścieżkę do rozpakowanego folderu:
            // Przykładowa ścieżka:
            // C:\\Users\\TwojaNazwa\\Desktop\\MojeRozszerzenie\\
            
            const path = 'Wprowadź/Pełną/Ścieżkę/Do/Folderu/Rozszerzenia';
            require('electron').remote.session.defaultSession.loadExtension(path)
                .then(({ id }) => console.log('Rozszerzenie załadowane ID:', id))
                .catch(err => console.error('Błąd ładowania rozszerzenia:', err));

Ta funkcja jest przeznaczona dla zaawansowanych użytkowników, którzy chcą testować własne rozszerzenia.
